# Getting Started with VS Code Extension

## Run the extension

- Press `F5` to open a new window with your extension loaded.
- Open a `.wl`, `.m`, or `.nb` file and save it to trigger shortening.

## Make changes

- Edit `extension.js` for activation and save-hook logic.
- Edit `transformer.js` to implement the `shorten()` transformation.

## Explore the API

- Open the full set of supported APIs in the [VS Code API documentation](https://code.visualstudio.com/api/references/vscode-api).
